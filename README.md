# get_clip_img
É um programa escrito em rust baseado no [imgur_up](https://github.com/srtopster/imgur_up), mas essa versão pega a imagem do clipboard do sistema para fazer upload.
> Utiliza [notifu](https://www.paralint.com/projects/notifu/) para notificações no windows.

> Não se esqueça de colocar seu client_id do imgur [aqui](https://github.com/srtopster/get_clip_img/blob/main/src/main.rs#L35)
## Quick start
```console
cargo build --release
target/release/get_clip_img.exe
```
